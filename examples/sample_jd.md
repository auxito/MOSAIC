# 示例 JD — 高级后端工程师

## 公司简介

一家快速发展的互联网公司，专注于内容推荐和个性化服务。

## 岗位要求

### 必须
- 3年以上 Python 后端开发经验
- 熟悉微服务架构，有服务拆分和治理经验
- 熟悉 Redis、MySQL 等存储系统
- 熟悉消息队列（Kafka/RocketMQ）
- 有高并发系统开发经验

### 优先
- 推荐系统开发经验
- Go/Java 经验
- Kubernetes 部署运维经验
- 机器学习工程化经验
- 大规模分布式系统设计经验

## 岗位职责
- 负责推荐系统核心服务的开发和优化
- 参与系统架构设计和技术选型
- 带领小团队完成项目交付
- 持续优化系统性能和稳定性

## 技术栈
Python, Go, Redis, MySQL, Kafka, Kubernetes, PyTorch
